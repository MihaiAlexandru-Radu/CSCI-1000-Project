# CSCI-1000-Project

## Usage

This project was primarily used as a practice for branching and merging in groups. The main product is the website, which should be accessible from this repository's description. From there you can click on each group member's personal page and learn more about them, also with the option to return to the main page if needed. 

The project also kept the original GitHub project which was used as a template for this project. You can open it by clicking on the last button on the home page.
